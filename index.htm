第一章 总则

   为了营造一个安全有效、秩序良好的实验室环境,达到“科学、规范、安全、高效”的目的,特制订本实验室手册，本手册适用于进入实验室内所有人员。

第二章  实验室安全制度

一、人员安全规范

  1. 进入实验室，必须穿着实验服。
  2. 进行危害物质、挥发性有机溶剂、特定化学物质或其他毒性化学物质等化学药品操作时，必须穿戴防护具（例如防护手套、护目镜等，防止化学药剂溅入眼内灼伤眼睛） 。
  3. 实验室内不得穿拖鞋、凉鞋进入，需穿不露脚趾的满口鞋，不得穿着短裤、短裙，严禁披发。
  4. 操作高温（低温）实验，必须穿戴防护手套。

二、饮食规定

  1. 实验室内严禁饮食、饮水。
  2. 实验室的冰箱、冰柜和储藏柜禁止储藏食物。

三、区域规范

  1. 实验室内保持安静，严禁大声喧哗、吵闹。
  2. 未经实验室管理人员允许不得将外人带进实验室。
  3. 不得在实验室做与实验无关的事情。
  4. 实验期间严禁长时间离开实验室。
  5. 进行危险实验，严禁单独进行实验，以保实验安全。
  6. 夜间进行实验需向导师报备。

四、实验室安全规则

（一）进入实验室前应知

  1. 保持实验室门和走道通畅。
  2. 了解实验室及建筑物的所有安全出口，熟悉逃离路线、紧急冲淋器、急救箱和灭火器的位置，并知道使用方法，应了解气总阀门、水阀门及电闸所在处。

（二）实验过程中应注意

1. 用电安全

1.1 使用电器设备（如烘箱、恒温水浴、离心机、电炉等）时，严防触电；绝不可用湿手或在眼睛旁视时开关电闸和电器开关。应该用试电笔检查电器设备是否漏电，凡是漏电的仪器，一律不能使用。

1.2 实验室电路及用电设备要定期检修，保证安全，决不“带病”工作。如有电器失火，应立即切断电源，用沙子或灭火器扑灭。在未切断电源前，切忌用水或泡沫灭火器灭火。

1.3 如发生人身触电事故，应立即切断电源，必要时进行人工呼吸，急送医院救治，并立即向实验室老师或负责人汇报。

2. 试剂安全

2.1 不得用手直接接触药品,更不得食用试剂。

2.2 使用可燃物，特别是易燃物（如乙醚、丙酮、乙醇、苯、金属钠等）时，应轻拿轻放，远离高温区域，防止发生火灾。

2.3 离开试剂瓶的试剂既不能放回原瓶,也不能随地丢弃,更不能拿出实验室,要放入对应的回收容器内。

2.4 易燃、易爆的化学药品要妥善分类保管，注意消防安全；做化学实验时要严格按照操作规程进行，谨防失火、爆炸等事故发生。

2.5 有毒物质应按研究所的规定办理审批手续后领取，使用时规范操作，用后妥善处理。

2.6 使用后的试剂（如强酸、强碱）不能直接倒在水槽中，要倒在指定废液桶中，集中处理。

3. 操作安全

3.1 阅读所用仪器设备操作规程，化学品的安全技术说明书，留意实验室与仪器上的各类警示标志等；熟知所用物品、设施和设备的潜在危险。

3.2 危险实验(进溅、爆炸、剧烈放热、挥发、生成物有毒性等)需要佩戴护目镜、口罩等防护用品，实验过程中如有毒有害气体产生，需在通风橱内操作，做好室内通风。

3.3 严格遵守仪器操作规程和实验守则。挪动设备或调节设备参数前，应与相关人员沟通，不得随意拆装设备零配件，严禁离开正在运行的高速、高压、高温装置。

3.4 使用强酸、强碱，须轻拿轻放，防止溅出。若不慎溅在实验台上或地面，必须及时用湿抹布擦洗干净。如果触及皮肤或经口、鼻、眼进入体内，应按照《强酸强碱烧伤、中毒的应急救护》（附件一）处理。

4. 防火安全 4.1 实验室内严禁吸烟（包括室内、走廊、电梯间等）。

4.2 易燃易爆试剂

  对于易燃易爆试剂包括：爆炸品、压缩气体和液化气体、易燃液体、易燃固体及自燃物品和遇湿易燃物品、氧化剂和有机过氧化物、腐蚀品、放射性物品以上试剂，应做到领用后暂时不用的危险化学品应存放于固定存放点；并做好防挥发、防泄露、防火等安全措施。

4.3 高温高压设备

  高温高压设备使用过程中应严格执行操作规程，保证安全运行；使用者应对使用状况进行经常性检查，发现问题应立即处置。

4.4 掌握灭火器使用方法（二氧化碳灭火器）

  1. 将灭火器提起，移至火场上风向3-5米处；
  2. 拔掉保险销，将喇叭筒金属导管与瓶体呈70-90度；
  3. 由于二氧化碳由液体变成气体需要大量吸热，因此在没有有防护的的情况下，不要直接触摸喇叭筒和喷射胶管，应一手握住握把，一手托住瓶体底部，对准火源根部进行左右喷射。如果在较狭小的区域使用，尽量在使用完迅速撤离，防止窒息。

（三）离开实验室请注意

  1. 离开实验室前脱下手套和实验服，做好个人清洁，填写实验室记录表。
  2. 应关闭非运行状态中的仪器设备。
  3. 将室内检查一遍，应将水、电、气的阀门关闭，门窗锁闭。

第三章 实验室试剂、仪器管理制度

一、化学试剂的储存领用

  1. 实验室试剂设立1名管理人员。
  2. 化学试剂应分类储存，存放防潮、避光、通风的试剂柜中，并录入EMEP管理系统，通风橱内不得储存化学试剂。
  3. 领用化学试剂时，需在EMEP管理系统填写，并更新剩余量，药品领用后应及时放回试剂柜。
  4. 熟悉所使用的化学试剂的特性和潜在危害。
  5. 所有使用容器都要标记容器内化学试剂名称、使用人及日期。
  6. 对于在储存过程中不稳定或易形成过氧化物的化学试剂需加注特别标记。
  7. 在实验室内不得储存大量易燃溶剂，用多少领多少。
  8. 装有腐蚀性液体容器的储存位置应当尽可能低，并加垫收集盘，以防倾洒引起安全事故。
  9. 腐蚀性化学品、剧毒化学品、有机过氧化物、易自燃品和放射性物质应分隔保存，标签上标明购买日期。
  10. 剧毒化学品需要特殊储存条件，未经允许不得在实验室储存剧毒药品。

二、实验仪器管理使用制度

  1. 实验室仪器设立一名全局管理人员，每台仪器各设1位管理人员。
  2. 所有仪器设备（包括新增、更换）建立档案录入EMEP管理系统，包括设备名称、生产厂家、型号、实验室验收日期、管理人员、仪器说明书、配件信息等。
  3. 爱护仪器设备，如发现仪器或相关配件丢失或损坏，必须及时向实验室管理人员汇报；同时在实验室记录本和EMEP管理系统如实登记。
  4. 凡损坏而不能维修或失去维修价值的仪器设备，向实验室管理人员汇报，经研究所批准后执行报废流程； 在EMEP管理系统登记。
  5. 规范仪器使用，使用时保持仪器整洁，保持仪器状态前后一致。
  6. 实验仪器管理人员如有变动，及时办理好仪器管理交接手续。并汇报实验室管理人员，并在EMEP管理系统登记。
  7. 实验室内仪器不得私自借出，特殊需要必须借用的需经过实验室管理人员及团队负责人批准后，在本实验室使用；借用人员需填写实验室仪器设备借用表，如有损坏，照价赔偿。

三、实验耗材管理使用制度

  1. 实验耗材包括实验材料（生物质、土壤、植物等材料）和耗材类（玻璃器皿、陶瓷金属容器、文具、纸类、塑料制品等），分别设1名管理人员。
  2. 管理人员负责实验耗材购买、验收、登记及库存管理。
  3. 实验耗材建立耗材目录录入EMEP管理系统，包括实验材料（名称、类别、所有者、数量、采集日期、存放位置和管理人员等）、耗材类（名称、CAS号、类别、规格、数量、购买日期和管理人员等）。
  4. 耗材类有所消耗或损耗，应及时更新余量。

第四章 卫生管理制度

  1. 进入实验室的所有人员，必须整洁、文明、肃静，必须遵守实验室的各项规章制度。
  2. 实验人员在实验过程中，要注意保持室内卫生。实验结束后，必须及时做好清洁整理工作，实验人员必须将工作台、仪器设备、器皿等清洁干净，并将仪器设备和器皿按规定归类放好，不能随意放置。所有实验所产生的废物应及时放入废物箱内，并及时处理。
  3. 若开展的实验（如土壤、植物样品前处理，生物炭制备）会污染整个实验室，试验结束后，实验人员必须对整个实验室（仪器、试验台及地面）进行清扫。
  4. 实验室每周设一名值日生，负责公共区域卫生和卫生监督，值日表根据近期参与实验的同学进行调整。
  5. 实验室内各种设备、材料摆放整齐，与实验无关的物品禁止带入、存放在实验室。
  6. 实验室为保持室内地面、实验台、设备和工作环境的干净整洁，每人保持个人试验台整洁，每月一大扫除的卫生制度，要求全体人员参加。大扫除应不留卫生死角，将实验室内的仪器设备、各人实验台架、凳和各种设施摆放整齐，擦拭干净，保持无污渍、无灰尘。
  7. 离开前必须做好清洁卫生，关好门窗、水龙头，断开电源。

第五章 实验室仪器试剂耗材购买

一、实验室仪器试剂耗材的采购

  1. 各同学应根据实验实际需要，提交《仪器试剂耗材采购清单》（附件二）至导师处审核，审核通过后提交至各管理人员。
  2. 各同学提交采购计划时应遵守“计划充分，杜绝浪费”的原则。

二、仪器试剂耗材的验收

      仪器试剂耗材到货后由实验室管理员与相应的同学共同进行验收，验收无误后储存至实验室各保存柜内并录入EMEP管理系统。

第六章 人员培训与考试制度

  1. 进入实验室前，自行熟知上述实验安全规范手册，参加实验室全体管理人员组织的培训。
  2. 培训结束后后由实验室管理人员进行考核，考核内容包括但不限于口试、笔试、现场操作演练等。考试合格后签订实验安全规范手册知悉书，可进入实验室开展实验。
