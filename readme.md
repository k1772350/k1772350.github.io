#食物链
Carlin等研究了可能以小型哺乳动物、两栖动物、蠕虫和昆虫为食的捕食鸟类胃肠道中MPs的积累情况，结果表明，具有多层次营养结构的顶级捕食者的微塑料含量更高。Lwanga等人（2017年）测试了传统玛雅家庭花园的土壤、蚯蚓尸体、鸡粪、作物和鸡胗中的微塑料和大塑料的丰度，并首次提供了塑料从土壤迁移到鸡的证据。这些事实意味着微塑料已经渗透到土壤食物网中，并将到达其他环境，形成一个 "微塑料循环"。不仅如此，它最终会对人类产生影响。Lwanga等人发现，在墨西哥，人们通过吃鸡，每人每年可能摄入840个塑料微粒。
土壤中的微塑料也可以通过食物链发生传递、 富集,带来健康风险。 Huerta Lwanga 等首次报道 了微塑料在庭院土壤—蚯蚓和土壤—鸡食物链中的 传递,发现微塑料从土壤到蚯蚓粪的富集系数高达 12.7,而从土壤到鸡粪的富集系数更是高达 105。
而在土壤环境中, Lwanga 等人发现了在陆地食物链中转移塑料 碎片的现场证据, 即在家庭花园土壤-蚯蚓-鸡 的食物链中, 土壤、蚯蚓粪、鸡粪和鸡胃中均检测 到微塑料, 微塑料富集在蚯蚓粪及鸡粪中。此外, 微塑料在动物体内的富集程度仍需更 s 深入的研 究。一项野外调查首次发现可食用陆生蜗牛体内 存在微塑料, 虽然丰度较低, 但表明人类或者其 他动物可能通过取食蜗牛而摄入微塑料。在陆 生鸟类的研究中, Carlin 等人发现赤肩鵟、鱼鹰等 8 种猛禽胃肠道中含有微塑料, 且以纤维塑料居 多。作为食物链中高营养级动物, 猛禽以鼠类 等小型哺乳动物、蛇、蜥蜴及两栖动物为食, 微塑 料污染的水及猎物可能是猛禽摄入微塑料的重要 来源, 其中营养级迁移机制需进一步研究。

#空气河流
风、河流和生物效应不仅是土壤微塑料的来源，也是微塑料向其他环境迁移的途径。土壤中超过60%的塑料颗粒最终会扩散到河川流域，污染水体。因此，土壤是MPs的 "仓库"，是周围环境中塑料颗粒的来源。对从陆地环境迁移到海洋的微塑料通过河流下沉并在海洋中迁移的追踪技术仍在发展有研究表明,陆地环境中的微塑料 可以通过风力作用迁移至大气中[78]。通过风力作用, 微塑料在空气中至少可移动 95 公里, 同时据推测 微塑料可以在地面上较小的空间尺度上移动。 Rezaei 等人的研究表明风蚀对微塑料在陆地环境中 的迁移起到关键作用[50], 同样雨水径流对土壤表层 微塑料的迁移也起到重要作用
#地下水
以前的研究发现，当微塑料尺寸小于土壤孔隙尺寸时，微颗粒可能通过土壤孔隙和裂缝向下移动。Zhou等人在长江沿岸河岸土壤的29个地点进行了实地研究，结果显示，底土（10-15厘米）的微塑料含量高于表土（0-5厘米），小尺寸的微塑料（<200微米）约占70%。作者还证实，微塑料污染与降水密切相关，表明微塑料通过土壤孔隙的沥滤向下运输。Goeppert和Goldscheider发现，在德国上莱茵河谷，1-5微米的微塑料颗粒可以在3.1至200米的冲积含水层中大量迁移。利用INCA-污染物模型，Nizzetto等人开发了一个模拟工具，并预测土壤中较小的微塑料（<0.2毫米）最终会被输送到浅层地下水。到目前为止，地下水中的微塑料污染被忽略了，但已经检测到12个颗粒L-1，这表明微塑料从土壤到地下水的输送确实发生了（Panno等人，2019）。在地下水井的原始饮用水中检测到低浓度的微塑料（0-7颗粒m-3），尺寸在50至150微米之间。在南印度的地下水（尺寸：0.12-2.50毫米）和地表水（尺寸：0.34-4.30毫米）中发现了包括聚酰胺、聚酯、聚丙烯、聚乙烯、聚氯乙烯和纤维素等主要聚合物类型的微塑料（高达19.9颗粒L-1），并显示出对重金属的吸附能力。此外，地下水也容易受到来自地表水补给的微塑料污染，特别是来自污水处理厂和工业的废水。例如，在三个污水处理厂排放的荷兰河流地表水中检测到67至11,532粒/立方米的高浓度微塑料（Mintenig等人，2020），而根据全球50项研究，微塑料的中位浓度为2.75粒/立方米。到目前为止，大多数关于地下水和地表水中微塑料的研究报告了少数聚合物类型和较大的颗粒，而较小的颗粒（如<100-300微米）则更为丰富。此外，由于采样和量化技术的限制，目前的研究仍然缺乏关于微塑料从土壤进入地下水体的坚实实验数据。
微塑料进入地下水和水生环境的来源有人工排水、大量大孔和发生地表径流的农业土壤中，这种流失的风险预计会大得多。目前，地下水的微塑料污染很低，但已显示出达到12个颗粒l-1的水平，表明确实发生了转移。
除了在土壤内迁移外，土壤中的微塑料还可以通过风、灰尘、侵蚀和地表径流等自然或人为现象迁移到周围的环境介质，如空气和水。由于风和气流，装载在土壤表面的微塑料，尤其是微纤维可以提升到空气中，在空气中悬浮一段时间。土壤中的微塑料也可能进入地下受体，例如含水层。最近发现，干湿循环数与微塑料的渗透深度呈正线性相关，根据天气数据，中国100个城市的5年渗透深度平均估计为24.1米（范围48.7-42.347米）。使用INCA-Contaminants模型，Nizzetto等人开发了一种模拟来确定从土壤转移到泰晤士河的微塑料部分。他们的结果表明，土壤中超过60%的微塑料最终会迁移到河流集水区并污染水环境。
#深层土壤
发现了纤维通过未知机制向下易位的证据。最近的研究人员在表土和深层土壤中都检测到微塑料。这些结果表明，微塑料可以在土壤中垂直移动。正如Cey等人所报道的那样，平均直径为3.7μm的微塑料可以通过淋溶向下移动到70厘米以上的深层土壤。甚至有人预测，微塑料可能会在浸出的帮助下最终进入浅层地下水
积累在陆地环 境中的部分微塑料则在植物根系、生物和机械扰动 作用下发生迁移。
